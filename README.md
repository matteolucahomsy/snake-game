# Snake Game 🐍

A simple Snake game built with Python and Pygame.

## Features
- Snake movement
- Food spawning
- Score system
- Collision detection
- Game over screen
- Increasing challenge over time
- Snake move automatically

## Technologies Used
- Python 3.12
- Pygame

## Installation

Clone the repository:

```bash
git clone https://github.com/matteolucahomsy/snake-game.git
```
### Run the game
```bash
python main.py
```
